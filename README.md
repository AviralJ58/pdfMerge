# PDF Merger

Simple program to merge the PDF/PPT/PPTX in the specified folder with given output filename.
If the folder has PPT/PPTX then it will be automatically converted to PDF and can be merged. The converted PDFs will be deleted from the directory when the process is completed.

**Modules used**: PyPDF2

The user has **two options**:
 1) Merge all the PDF/PPT/PPTX in the directory.
 2) Merge selective files by choosing from a list of PDF/PPT/PPTX available in the directory in specified order.
